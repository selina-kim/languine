**Short Description**\
When language learners encounter a new word, it can be difficult to commit that word to their vocabulary or remember to save it for
later. Our vision is to have an application that given a word in a target or known language (through text or speech), would 
automatically create a card that can be saved to a deck with the translation of the word, and examples of the word used in context, 
so the user can focus on memorizing vocabulary rather than card creation. We also would like to support models and/or games for 
deck revision using an algorithm, uploading and exporting decks so they can be shared with friends, etc. And provide multi-language
support. Applications do currently exist for card revision, but none of them generate cards/decks for you automatically, and many
are hidden behind paywalls.
